# Algorithme de Kruskal

## Description simple

Pour décrire simplement l'algorithme de Kruskal, prenons l'exemple d'un facteur qui doit livrer des colis à plusieurs maisons (noeuds). L'objectif du facteur est de déposer tous les colis en prenant le moins de temps possible (coût) en utilisant les chemins (arêtes) disponibles.
